Kasthamandap Dental Clinic Online Patient Management System

Web App Development

This was my first "big" project. It was my Final Year project for my Bachelor's degree. I used my dad's clinic as the client to create a web application using object-oriented PHP, which I had only managed to learn at the start of that year. But I picked it up quickly with the help of a friend, and was able to develop a pretty solid web app to show information about the clinic as a generic informative website. On top of that, I was also able to create a patient management application to take appointments, store and view patient records and so on. I wanted to implement it but I wasn't sure how well it would work.

  • Date: September 2013 - May 2014
  • Client: Kasthamandap Dental Clinic
  • Category: Web Development, Final Year Project
